## New public contracts

- `service_templates`: implemented Admin-managed service definitions linked to `service_categories`; operational counts are derived.
- Protected RPCs: upsert/duplicate/set-status/archive service template; restore/permanently-delete/empty-trash with allowlists and auditing.
- `admin-invite-account` Edge Function: implemented validated email, display name and fixed `USER`/`WORKER` role; sends an expiring Supabase invite.
- `cancel_booking`, manual wallet top-up/review, push subscription, private support attachment, notification command, aggregate, and structured onboarding RPCs are generated in the shared database contract.
- Provider capability response exposes only the Google OAuth availability flag. Payment provider capability flags and PayMongo client/function paths are retired.
